Projekt: Django e-shop (1. faze)

Popis:
- Zakladni Django projekt a aplikace
- Datovy model: Category, Product, Customer, Order, OrderItem
- Admin napojen na modely

Pozadavky:
- Python 3.x
- Balicky v requirements.txt

Spusteni:
1) Aktivujte venv
2) Nainstalujte zavislosti: pip install -r requirements.txt
3) Migrace: python manage.py migrate
4) Spusteni serveru: python manage.py runserver
5) Admin: /admin

Poznamka:
- SQLite databaze je v souboru db.sqlite3
